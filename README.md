# 🎓 Student Management System

A C++ based Student Management System developed to manage student records using fundamental programming concepts. This project demonstrates record management operations such as adding, viewing, searching, deleting, and updating student information through a menu-driven interface.

---

# 📖 About

The Student Management System is designed to help understand:

- Data Management
- Arrays
- Loops
- Conditional Statements
- Searching Techniques
- Record Handling
- Software Development Workflow

This project is part of my journey to strengthen programming and problem-solving skills through practical development.

---

# 🚀 Current Features

## ✅ Add Student

Allows users to:

- Add student name
- Add roll number
- Store student records

---

## ✅ View Students

Displays all student records in a structured format.

Information displayed:

- Student Name
- Roll Number

---

## ✅ Search Student

Allows users to search student records using roll numbers.

Features:

- Fast record lookup
- Displays matching student information

---

## ✅ Delete Student

Allows users to remove student records using roll numbers.

Features:

- Record deletion
- Automatic record reorganization
- Updated student list management

---

## ✅ Update Student

Allows users to update existing student records using roll numbers.

Features:

- Update student name
- Update roll number
- Modify stored records easily

---

## ✅ Menu Driven Interface

Simple and user-friendly console interface for managing student records.
