# Student Management System

A C++ based project to manage student records using basic programming concepts.

## Features

### Current Features
- Basic Menu System
- Add Student Option
- View Students Option
- Search Student Option
- Exit Option

### Planned Features
- Store Student Name
- Store Roll Number
- View All Students
- Search Student by Roll Number
- File Handling
- Delete Student Record

## Technologies Used
- C++
- Loops
- Conditional Statements
- Switch Case
- Arrays
- File Handling (Planned)
